# Realtime Chat Application

## Công nghệ sử dụng
- ASP.NET Core
- SignalR (Realtime)
- HTML, CSS, JavaScript
- JWT Authentication
- SQL Server

## Chức năng chính
- Đăng nhập, xác thực người dùng
- Chat 1 - 1 theo thời gian thực
- Chat nhóm
- Tạo nhóm, thêm / xoá thành viên
- Lưu lịch sử tin nhắn

## Mục tiêu
- Hiểu cách hoạt động của realtime communication
- Làm quen với SignalR và mô hình Client - Server

## Ghi chú
- Project đang trong quá trình phát triển
- Chỉ sử dụng cho mục đích học tập
